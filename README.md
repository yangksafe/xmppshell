一个简单的xmpp机器人 

shell远程管理工具

主要用于管理我的服务器

方便在所有服务器上批量执行命令

目前的功能

——

自动连接到xmpp服务器

密码验证功能

接收以验证用户的消息

作为执行shell命令执行

——

运行在python3上

python -m pip install urllib3==1.25.11
